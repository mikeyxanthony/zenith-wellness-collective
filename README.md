The site uses placeholder images from Unsplash. Replace with your custom images:


### Update Colors

Modify CSS variables in `styles.css`:

```css
:root {
    --cream: #F8F6F0;      /* Your cream color */
    --charcoal: #2C2C2C;   /* Your charcoal color */
    /* ... etc */
}
```
